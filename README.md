# Voryn Works – Static Site Clone

A minimal React + TypeScript + Vite project that replicates the static landing page of https://atum.works. The codebase is a clean, copyright‑free rewrite with custom theme variables, a simple hero component, and Tailwind CSS styling.

## Features
- Dark theme with custom `--font-mono` and `--color-aw-gray-light` CSS variables.
- Responsive layout: Navbar, Hero, Footer.
- Tailwind‑CSS utilities for animations and glass‑morphic effects.
- TypeScript support, ESLint, and Vite HMR.

## Getting Started

```bash
git clone https://github.com/rxz33/voryn-atum-clone.git
cd voryn-atum-clone
npm install
npm run dev
```

Open http://localhost:5173 to view the site locally.
